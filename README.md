# Simple Interest Calculator

## Project Description
This project is a **Simple Interest Calculator** created as part of the IBM Introduction to Git and GitHub course.

## How it Works
The calculator computes **Simple Interest** using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Input Parameters
- Principal amount
- Rate of Interest
- Time period (in years)

## Output
- Calculated Simple Interest

## Technology Used
- Bash Shell Script
